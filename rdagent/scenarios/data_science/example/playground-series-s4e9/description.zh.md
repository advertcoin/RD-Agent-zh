## 评估

### 均方根误差 (RMSE)

提交的得分基于均方根误差。RMSE 的定义如下：

$$
\mathrm{RMSE} = \left( \frac{1}{N} \sum_{i=1}^{N} (y_i - \hat{y}_i)^2 \right)^{\frac{1}{2}}
$$

其中 $\hat{y}_i$ 是每个实例 $i$ 的预测值，$y_i$ 是原始值。

### 提交文件

对于测试集中的每个 `id`，您必须预测汽车的 `price`。文件应包含表头，格式如下：

```
id,price
188533,43878.016
188534,43878.016
188535,43878.016
etc.
```

## 关于表格 Playground 系列

表格 Playground 系列的目标是为 Kaggle 社区提供各种相对轻量级的挑战，可用于学习和提高机器学习和数据科学不同方面的技能。每次竞赛的持续时间通常只有几周，根据挑战的不同可能会更长或更短。挑战通常使用相对轻量级的数据集，这些数据集是从真实世界数据合成生成的，并将提供机会快速迭代各种模型和特征工程思路，创建可视化等。

### 合成生成的数据集

在 Playground 竞赛中使用合成数据使我们能够在拥有真实世界数据（具有命名特征）和确保测试标签不公开之间取得平衡。这使我们能够举办比过去更有趣的数据集竞赛。尽管合成数据生成仍然存在挑战，但现在的技术水平比两年前我们开始表格 Playground 系列时要好得多，目标是生成 artifact 少得多的数据集。请随时向我们提供有关不同竞赛数据集的反馈，以便我们继续改进！

## 数据集描述

本次竞赛的数据集（训练集和测试集）是从在[二手车价格预测数据集](https://www.kaggle.com/datasets/taeefnajib/used-car-price-prediction-dataset)上训练的深度学习模型生成的。特征分布接近但不完全相同于原始数据。您可以随意将原始数据集作为本次竞赛的一部分，以探索差异以及查看将原始数据纳入训练是否能提高模型性能。
