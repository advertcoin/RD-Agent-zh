# 文件夹结构设计

## 概念

当我们优化解决方案时，我们有以下策略。
- `draft`: 创建新解决方案
- `idea`: 提出改进解决方案的想法
- `merge`: 合并不同的解决方案

优化是一个漫长的过程；我们可能在不同策略之间切换。这些策略被称为路由器。
- `router`: 在不同策略之间路由的元策略。路由器可能有不同的实现。

工具：
- `select`: 提供供其他策略或步骤使用的功能。
  1) `submit`: 在结束优化之前，我们必须选择唯一的一个解决方案进行提交。
  2) `expand`: 我们可以选择一个点开始下一次扩展。

## 建议的文件夹结构

因此建议的文件夹结构是：
```
- router/
- idea/
  - samll_step(or refine?).py
  - normal.py
- draft/
- merge/
- select/
  - expand.py
  - submit.py
```
